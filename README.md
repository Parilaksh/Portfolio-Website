# Personal Portfolio Website

This is a personal portfolio website created using HTML and CSS.

##  Features
- Responsive layout
- Navigation bar
- About section
- Hobby section (Bike Riding)
- Projects section (3 projects displayed using CSS Grid)
- Contact form
- Footer with social links
- Google Fonts integration

##  Technologies Used
- HTML5
- CSS3
- Google Fonts

##  How to Run the Project

1. Download the project folder.
2. Extract the ZIP file.
3. Open the folder.
4. Double click on `index.html`.

The website will open in your default web browser.

No additional installation or configuration is required.
